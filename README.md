# ZEDI_website
A website to access ZEDI data from a segregated file system

WORKS FOR ONE NODE ONLY

Steps to run:

STEP 1: Download the whole folder
STEP 2: Set directory variable in python converter_1s and converter_5m to the place where 1 second and 5 minute data are stored
STEP 3: Install dependencies
STEP 4: Run the python file connector.py
STEP 5: Click on the link python gives you on running the program
STEP 6: Fill details and file is generated
